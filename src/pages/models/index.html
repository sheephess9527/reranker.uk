bge-reranker

Open weightsSelf-host

Open-weight rerankers from BAAI. A strong English baseline (bge-reranker-base) and excellent multilingual models (bge-reranker-v2-m3). The default choice when you want to self-host for free.

Cohere Rerank

Hosted APIMultilingual

The most mature hosted rerank API, with consistent multilingual quality, a generous free tier and SDK support across Python, Node, Java and Go.

Jina Reranker

Open + HostedTiny variant

Unique in offering both a hosted API and open weights — including a tiny model small enough to run in the browser (which powers our demo).

Voyage Rerank

Hosted APIDomain-specific

Voyage AI's rerankers are optimised specifically for retrieval precision and offer domain-tuned variants for code and finance.

Open weightsBrowser xsmall

Apache 2.0 DeBERTa-v3 rerankers from mixedbread-ai. The xsmall variant runs in the browser and powers our demo.

How to choose

Open

bge-reranker

Best free self-hosted option

  • Zero per-call cost — runs on your own infra
  • Strong multilingual quality (v2-m3 covers 100+ languages)
  • Drop-in with sentence-transformers, LangChain, LlamaIndex
API

Cohere Rerank

Easiest hosted API, top multilingual quality

  • Official SDK for Python, Node, Java, Go — one-liner integration
  • Consistent multilingual quality across 100+ languages
  • Generous free tier; mature, production-proven API
Open + API

Jina Reranker

Maximum flexibility — API, self-host, or browser

  • Choose hosted API or open weights — same model family
  • Tiny variant runs in the browser (no server needed)
  • Free tier, no credit card required to start
API

Voyage Rerank

Top retrieval precision, domain-specific variants

  • Tuned specifically for retrieval quality, not just classification
  • Domain-specific models for code search and finance
  • Lowest per-1k-doc price among the hosted APIs
Open

mxbai-rerank

Apache 2.0 open weights, browser-runnable xsmall

  • Permissive Apache 2.0 licence — use commercially without restrictions
  • xsmall variant runs in the browser (powers our live demo)
  • Highest BEIR score in this table at the large size
